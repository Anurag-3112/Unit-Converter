# Features

## Core Features

- Real-time unit conversion
- Supports multiple unit categories
- Accurate conversion calculations
- Interactive and beginner-friendly UI
- Responsive design for all devices


## Supported Conversions

### Length
- Meter ↔ Kilometer
- Centimeter ↔ Meter
- Mile ↔ Kilometer
- Inch ↔ Centimeter

### Weight
- Gram ↔ Kilogram
- Kilogram ↔ Pound
- Pound ↔ Ounce

### Temperature
- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin


## UI Features

- Modern responsive design
- Smooth user interactions
- Clean typography and spacing
- Mobile-friendly layout
- Interactive dropdown selectors

# How It Works

1. Select a conversion category
2. Choose input and output units
3. Enter the value to convert
4. Instantly view the converted result
